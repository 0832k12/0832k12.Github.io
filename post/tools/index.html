下载工具
①
NeatDownloadManager

NeatDownloadManager是一款简洁、高效的下载管理软件，
它可以：
‎✔️加快您的下载速度，达到您的总可用带宽。‎
‎‎✔️使用优化的动态细分算法。‎
‎‎✔️根据下载状态（完整、不完整）和文件类型（视频、文档、。。。） 组织您的下载‎
‎‎✔️支持高频、高频和FTP协议。‎
‎‎✔️支持 HTTP 代理和袜子协议。‎
‎‎✔️支持 HTTP 身份验证和代理身份验证（基本、文摘、NTLM）‎
‎‎✔️具有暂停/恢复功能，可以恢复崩溃的下载。‎
‎‎✔️有浏览器扩展，可以发送下载链接到它，并帮助您从任何Web网站下载视频/音频。‎
‎‎✔️即使下载正在进行中，也允许您在下载时设置带宽限制。‎
‎‎✔️即使下载正在进行中，也允许您在下载时设置最大连接号。‎
‎‎✔️可以下载所有。ts文件的HLS视频，并在结束时合并到一个单一的。ts文件的所有段。‎
‎可以续订过期的下载。‎
‎‎✔️工作与拖放（只为Mac，观看‎‎视频剪辑‎）
来自OneDrive云盘的文件需要32线程下载。
